# Ecological Sustainability & Data Centers — Dashboard v3.3.1

Live: https://cjimmylin.github.io/cei-sustainability-dashboard-v331/

Static dashboard accompanying the paper *Ecological Sustainability and Data
Center Environmental Impact: A Global Governance Landscape* (Lin, 2026).

Audits 3,044 AI governance statements worldwide for substantive engagement
with data-center environmental impact. Headline finding: only 0.95% of the
corpus reaches Tier A engagement — a universal governance–reality gap, not
an Anglophone privilege gap.

## What's in this release (v3.3.1)

This is a **patch release** with no user-visible changes from v3.3. The
patch closes an internal generator-vs-output drift that had accumulated
since S17. See `CHANGELOG.md` for details.

For the v3.3 release (the first with onboarding narrative + per-tab intro
hooks + accessibility audit at 0 violations), see:
https://cjimmylin.github.io/cei-sustainability-dashboard-v33/

## Reproducibility

Generated by `scoring/generate_dashboard.py` in the parent project at
`CEI-AI-Statements/SUSTAINABILITY/`. As of v3.3.1, the generator is fully
self-reproducing: re-running it produces byte-identical output. See
`METHODS/20260413b-sustainability-s19-generator-reconciliation.md`.
